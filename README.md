File Handling Utility in Java
A lightweight, console-based Java application that demonstrates fundamental file I/O operations. This utility provides a clean example of creating, writing, reading, and appending text files using modern Java best practices.

🚀 Features
File Writing: Dynamically creates a text file and writes structured text data.

File Reading: Efficiently streams and prints content from a text file to the console line-by-line.

File Appending (Modification): Appends new logs or content to an existing file without overwriting the original data.

Resource Management: Uses Java's modern Try-with-Resources statement to ensure automated, safe resource closing and avoid memory leaks.

Buffered Performance: Implements BufferedWriter to minimize disk I/O operations.

🛠️ Tech Stack & Prerequisites
Language: Java (JDK 7 or higher recommended)

IDE: Any Java-supported IDE (VS Code, IntelliJ IDEA, Eclipse) or standard Terminal.
